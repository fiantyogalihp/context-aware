PANDUAN MOBILE JKN

AKUN MOBILE JKN

  • Panduan untuk login ke MOBILE JKN

    Berikut merupakan cara untuk pengguna MOBILE JKN untuk masuk kedalam aplikasi MOBILE JKN

    1. Buka Aplikasi Mobile JKN, maka akan tampil halaman awal. Pilih Masuk/Daftar yang terletak pada pojok kiri atas
    2. Selanjutnya pilih Masuk
    3. Pilih jenis Identitas, masukan NIK/NOKA sesuai jenis identitas yang dipilih, masukan Password dan Captcha, lalu klik MASUK.

  • Panduan untuk login menggunakan FINGERPRINT atau FACE RECOGNITION

    Berikut merupakan cara pengguna MOBILE JKN untuk masuk kedalam aplikasi menggunakan fitur FINGERPRINT atau FACE RECOGNITION

    1. Untuk melakukan login Biometrik, klik tombol sidik jari disamping tombol Sign In. Login biometric ini hanya bisa dilakukan apabila sudah diaktifkan, dengan cara login terlebih dulu lalu masuk ke menu profil
    2. Ubah ke ON pada pilihan Login dengan Biometrik lalu akan muncul syarat dan ketentuan aktivasi biometric login, setuju dan lanjutkan. Kemudian masukan password login user tersebut
    3. Klik Verification button untuk melanjutkan
    4. Masukan password untuk mendaftarkan biometrik
    5. Apabila berhasil status login dengan biometric akan berubah ke ON
    6. Selanjutnya login sudah dapat dilakukan menggunakan biometric yang terdaftar pada perangkat tersebut, baik fingerprint maupun face recognition

  • Panduan untuk mengubah kata sandi yang lupa

    Lupa kata sandi merupakan fitur yang dikhususkan untuk pengguna MOBILE JKN yang lupa password akun

    1. Pada halaman login, plih fitur lupa kata sandi
    2. Setelah itu masukkan NIK, Captcha anda kemudian lanjut dengan klik “Lanjut”
    3. Kemudian pengguna memilih opsi(SMS/E-Mail/Pertanyaan Data Pribadi) untuk mendapatkan passwordnya kembali

  • Panduan untuk mendaftarkan diri anda menjadi pengguna MOBILE JKN

    Berikut merupakan tata cara yang dilakukan untuk mendaftar sebagai pengguna MOBILE JKN

    1. Langkah pertama yang anda lakukan saat masuk kedalam aplikasi MOBILE JKN yaitu dengan tap tombol daftar
    2. Setelah form pendaftaran ditampilkan, isi form pendaftaran berdasarkan sesuai kebutuhan yang diminta :
        a. Input Nomor Induk Kependudukan
        b. Input Nama
        c. Input Tanggal Lahir
        d. Input Captcha
    3. Setelah sudah tap verifikasi data, lalu anda akan dilanjutkan pada form lanjutan pendaftaran
    4. Tap input nomor handphone, input nomor hp lalu tap kirim kode verifikasi
    5. Muncul halaman syarat dan ketentuan, tap checklist saya setuju, tap selanjutnya
    6. Pada halaman verifikasi OTP No. Handphone, kode otp akan dikirimkan ke handphone anda , input kode OTP ke form verifikasi OTP
    7. Setelah semua berhasil di input, tap registrasi untuk menyelesaikan registrasi Anda akan diarahkan ke halaman utama tetapi status anda belum terdaftar menjadi peserta JKN

  • Cara mengubah pin

    Berikut merupakan tata cara yang dilakukan untuk merubah pin anda

    1. Pada menu home, arahkan view anda ke bagian profile, kemudian tap profile
    2. Kemudian ketika anda sudah masuk ke menu profile, tap ubah pin
    3. Masukan PIN anda saat ini, kemudian tap verifikasi
    4. Masukan PIN baru, tap verifikasi lalu tap setuju
    5. Kemudia muncul pop up bahwa pin sudah sukses diperbaharui

  • Panduan mengubah kata sandi

    Berikut merupakan fitur yang anda gunakan untuk merubah kata sandi dari akun yang anda gunakan

    1. Pada menu home, arahkan view anda ke bagian profile, kemudian tap profile
    2. Kemudian ketika anda sudah masuk ke menu profile, tap ubah kata sandi
    3. Lalu silahkan Isi Form ubah kata sandi yang disediakan dengan mengisi password lama, password baru, konfirmasi password baru, dan captcha kemudian tap simpan
    4. Kemudian muncul pop-up bawah kata sandi berhasil diubah lalu anda akan re-login Mobile JKN

  • Panduan untuk menyimpan akun yang sebelumnya telah login

    Fitur ini merupakan fitur yang memudahkan anda untuk dapat melakukan login dengan cepat dan aman, fitur ini akan menyimpan data akun terakhir yang terintegrasi dengan aplikasi MOBILE JKN

    1. Setelah anda login MOBILE JKN, buka menu Profil
    2. Terdapat fitur Simpan Data Username, aktifkan dengan klik tombol Off/on di sebelahnya, tap untuk mengaktifkan
    3. Setelah berhasil mengaktifkan, pada halaman login akan otomatis terisi dengan no. kartu peserta terakhir yang diaktifkan fitur Simpan Data Usernamenya

  • Panduan untuk mengaktifkan fitur notifikasi

    Kegunaan dari fitur ini yaitu untuk mendapatkan notifikasi yang diberikan oleh aplikasi MOBILE JKN terkait pelayanan program jaminan kesehatan

    1. Silahkan masuk ke menu profil
    2. Setelah masuk ke menu profil, geser button di baris notifikasi untuk diaktifkan
    3. Fitur notifikasi anda berhasil diaktifkan

  • Panduan untuk melihat notifikasi yang pernah terkirim

    Fitur ini berfungsi untuk melihat notifikasi yang dikirim oleh aplikasi MOBILE JKN terkait layanan jaminan kesehatan

    1. Tap gambar lonceng untuk melihat dafat lis notifikasi pada Mobile JKN
    2. Disini anda dapat melihat semua notifikasi yang pernah dikirimkan oleh aplikasi MOBILE JKN

  • Panduan untuk mengeluarkan akun dari aplikasi MOBILE JKN

    Berikut merupakan langkah-langkah yang dilakukan untuk mengeluarkan akun dari aplikasi MOBILE JKN

    1. Dengan cara masuk ke menu lainnya lalu tap menu keluar
    2. Ketika muncul pop-up veriikasi Tap Setuju
    3. Ketika anda sukses untuk mengeluarkan akun dari aplikasi, anda akan dikembalikan ke landing page Mobile JKN
